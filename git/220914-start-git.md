## DAY 1 Git

기초 shell Command
ls 	: 해당 위치에 하위폴더를 볼 수 있음
ls -a 	: 숨김파일 까지 볼 수 있다. 
ls -l	: 하위폴더들의 속성을 볼  있다.
mkdir 	: 디렉토리 생성
touch 	: 파일을 생성을 할 수 있다.
mv 	: 파일을 이동을 시킬 수 있다
rm	: 파일을 삭제할 수 있다.
