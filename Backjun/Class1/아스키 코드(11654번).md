## 아스키 코드(파이썬)

### 문제
알파벳 소문자, 대문자, 숫자 0-9중 하나가 주어졌을 때, 주어진 글자의 아스키 코드값을 출력하는 프로그램을 작성하시오.


### 입력
입력으로 주어진 글자의 아스키 코드 값을 출력한다.

### 출력

첫째 줄에 A+B, 둘째 줄에 A-B, 셋째 줄에 A*B, 넷째 줄에 A/B, 다섯째 줄에 A%B를 출력한다.

### 예제 입력 1
>A



### 예제 출력 1

>65

### 예제 입력 2
>C


### 예제 출력 2
>67



```shell
ASC = input()

print(ord(ASC))
```

ord (): 아스키 코드를 반환해준다.

chr(숫자): 숫자에 맞는 아스키 코드를 반환한다.
